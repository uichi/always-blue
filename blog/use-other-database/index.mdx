database.phpの中に、$defaultで設定しているデータベースと同じような形式で、使いたいデータベースを設定する。

```
class DATABASE_CONFIG {
  public $legacy = array(
    'datasource' => 'Database/Mysql',
    'persistent' => false,
    'host' => 'localhost',
    'login' => 'cakephpuser',
    'password' => 'c4k3roxx!',
    'database' => 'my_cakephp_legacy_project',
    'prefix' => ''
  );
}
```

こんな設定をしたとすると、あとはモデルで呼び出してやるだけです。
