autoremoveを使っていらないパッケージ削除をすると、ランチャーが表示されなくなってしまいました…

どうやらランチャーを表示するためのパッケージ、compizまで削除された模様

色々調べてみると、ドライバを新しく追加したり、UnityやUIをカスタマイズした際に今まで表示されていたものが消えてしまうといったことがあるようです。

## 参考
https://toxweblog.toxbe.com/2017/10/01/ubuntu-no-icon-menu/
http://m12i.hatenablog.com/entry/2014/05/24/100539
https://kledgeb.blogspot.jp/2013/07/unity-4-unitycompiz.html

参考の一番上の記事と同じようにやってみると、ランチャーが表示されました。

```
$ sudo rm -r ~/.cache/compizconfig-1
$ sudo rm -r ~/.compiz
$ sudo rm -r ~/.Xauthority
$ sudo rm -r ~/.config/autostart
$ sudo apt install –reinstall ubuntu-desktop unity compizconfig-settings-manager upstart
$ sudo dconf reset -f /org/compiz/
$ setsid unity
```
