# filter-branchとは
filter-branchはコミットの書き換えを機械的に行ってくれるオプションです。
大量のコミット履歴にある自分のメールアドレス変更や特定のファイルを削除することもできます。

# メールアドレスを一括変更してみる

実際のコマンドはこちらです。

```
git filter-branch -f --env-filter "GIT_AUTHOR_NAME=[your name]; GIT_AUTHOR_EMAIL=[your email]; GIT_COMMITTER_EMAIL=[your email];" HEAD
```

`[]`を置き換えて使用してください。
