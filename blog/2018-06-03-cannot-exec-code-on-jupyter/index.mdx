---
jupyterをpipでインストールし、起動はできたもののコードを実行することができませんでした。

jupyterとIPythonをアンインストールして再度インストールします。

```
pip uninstall ipython
pip uninstall jupyter
pip install ipython
pip install jupyter
```

IPythonは前々から挙動が不調だったためもう一度インストールし直すことにしました。
