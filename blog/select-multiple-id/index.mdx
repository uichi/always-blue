特定のid属性はそのページ内で1つしか存在できないという決まりがあるため、同じidを複数用意して選択時に何かしらの挙動を与えることは基本的にはできませんが、フィルタを指定することで実行することができます。

フィルタは$("[id=id名]")と指定することでかけられます。

こんな感じで使います。

```
$(function(){
  $("[id=test]").click(function(){
  $(".ui.modal").modal('show');
  });
});
```
