```
sudo vim /boot/config.txt
```

でディスプレイ用の設定を追加できます。

```
hdmi_group=2
hdmi_mode=87
hdmi_cvt=1920 1080 60 3 0 0 0
```

これと同じように追記し、再起動させると設定が反映されます。

詳しくは参考を見てください。
