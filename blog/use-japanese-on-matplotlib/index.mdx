jupyter notebookでmatplotlibを使いグラフを描画すると、日本語を標準でサポートしてくれないためラベルやグラフのタイトルに文字化けが起きてしまいます。

そのため、日本語を使えるようにするための設定が必要です。

設定は簡単で、以下を追加するだけです。

```
import matplotlib
font = {"family": "TakaoGothic"}
matplotlib.rc('font', **font)
```
