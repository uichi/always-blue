## アクセス解析研究所
無料で高機能なアクセス解析ツール

### メリット
+ 無料でも広告無し
+ 有料でも比較的お手頃なお値段
+ ビジターの追跡が尋常じゃない(GPS、ホストのIPアドレス・プロキシ、国や都道府県などの位置情報、言語やタイムゾーンから推測できる地域まで追跡可能)

### デメリット
+ 無料版は31日間利用しないと解析処理が停止する
+ 解析しないまま31日間が経過するとIDが削除される
+ Googleアナリティクスに匹敵するとまでいわれているツールです。

## USER LOCAL
最近話題になってると噂のアクセス解析ツール

### メリット
+ 直感的な操作
+ 解析できるページ数、サイト規模に制限がなし
+ 性別・年齢・地域・業界などユーザーの属性情報を自動取得できる
+ 流入元を分析

### デメリット
+ 計測できるサイトが3つまで
+ GoogleアナリティクスやYahoo!アクセス解析のような細かい分析が不得意

## 忍者アナライズ
### メリット
+ 管理画面は見やすい
+ 用語辞典があるので初心者に優しい

### デメリット
+ 広告表示
+ 3G携帯分析機能がない
+ ace analyzer

### メリット
+ 設置が簡単
+ デメリット
+ 1つのメールアドレスで1サイトしか登録できない
+ 3G携帯分析機能がない

## RESARCH ARTISAN Lite

### メリット
+ Googleアナリティクスと同等の機能
ページ数、プレビューの条件はなし
+ + 広告無し
+ 半永久的にログを取得

### デメリット
+ 設置が難しい
+ 高機能であるがゆえに、初心者にはハードルが高い(使いこなせるようになると強力)
