Reactでよく利用されるライブラリを紹介します。

随時更新していく予定です。

# ルーティング

## React Router Dom

Reactでルーティングさせるための定番ライブラリです。これをいれておけばルーティングは安心です。

https://v5.reactrouter.com/

# UI/UX

## Material UI

ReactのUIフレームワークを調べると必ず出てくるライブラリです。開発も盛んで利用者も多くいます。

https://github.com/mui-org/material-ui

## React Bootstrap

ReactのBootstrap用UIフレームワークです。

https://github.com/react-bootstrap/react-bootstrap

# フォーム

## React Hook Form

Reactでフォームを作成するための高機能ライブラリです。公式のサイトがかなり充実しています。

https://react-hook-form.com/jp/

## Formilk

こちらもReactで簡単にフォームを作成するためのライブラリです。よく耳にするライブラリの一つです。

https://formik.org/

# SEO

## React Helmet

Reactのheadタグを設定するためのライブラリです。必ずと言っていいほどインストールします。

https://github.com/nfl/react-helmet

# リッチテキストエディタ

## Draft.js

Reactのリッチテキストエディタ用ライブラリは数多くありますが、Draft.jsがその中でもよく使われています。

https://draftjs.org/

# アニメーション

## react-spring

UIにリッチな動きをつけることができるライブラリです。積極的に開発も行われているようです。

https://react-spring.io/
