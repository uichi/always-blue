Reactでテキストフィールドに初期値を入れたい場合`value`を使うとテキストの入力ができなくなります。

初期値を入れたい場合は`defaulValue`を使うと編集可能になります。

例）
```
<input type="text" defaultValue={this.state.value} />
```
