`firebase init`を実行して設定を進めていたときに、タイトルのようなエラーが出てきました。

メッセージを読んでみると、ロケーションが設定されていないと記載されています。

なので、Firebaseの管理画面でプロジェクトの設定を開き、ローケーションを設定します。
ロケーションの設定は少々時間がかかります。

再び`firebase init`をして設定を進めていくと今度はロケーションのエラーが表示されるはずです。

こちらを参考に、設定を進めてみてください。
