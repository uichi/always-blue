Homebrewが入っていることを前提として進めます。
# pyenvのインストール
```
brew install pyenv
```

pyenvのインストールだけでは動作しないので、環境変数の設定をします。
```
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bash_profile
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bash_profile
echo 'eval "$(pyenv init -)"' >> ~/.bash_profile
source ~/.bash_profile
pyenv rehash
```

# Pythonをインストール
```
pyenv install 3.6.10
pyenv global 3.6.10
```

`python -V`でバージョンを確認してみると、インストールしたバージョンのPythonが表示されます。
