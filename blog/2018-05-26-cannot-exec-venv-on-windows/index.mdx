pythonのvenv起動させようとすると実行できず…
調べてみると、powershellでは通常状態でスクリプトの実行ができないとのことでした。

特定のポリシーを割り当てることで実行できます。

```
Set-ExecutionPolicy Unrestricted -Force -Scope CurrentUser
```
