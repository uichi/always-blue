自動起動を操作できるサービスの確認は、rc5.dディレクトリで確認できます。

```
ll /etc/rc5.d
```

自走起動をオンにしたい場合

```
systemctl enable サービス名.service
```

自動起動をオフにしたい場合

```
systemctl disable サービス名.service
```

サービスが自動起動設定を確認したい場合

```
systemctl is-enable サービス名.service
```
