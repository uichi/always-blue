方法はいろいろあるのでここで紹介することに固執する必要はないのですが、currrentメソッドを使うことで配列の初めの部分を取得することができます。

```
os = array(Ubuntu, Arch, Windows, CentOS)
echo current(os)
```

Ubuntuが表示されます。
