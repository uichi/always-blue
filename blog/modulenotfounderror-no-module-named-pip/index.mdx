Ubuntuを16.04から18.04にアップグレードした後に、pipコマンドを実行したところpipが壊れていました。

なので、pipをもう一度インストールし直して修復を試みます。

```
wget https://bootstrap.pypa.io/get-pip.py
python3 get-pip.py
```

これでpipコマンドが使えるようになります。

以前使っていたパッケージも再度インストールし直す必要があるのでご注意ください。
