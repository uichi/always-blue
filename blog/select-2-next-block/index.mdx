隣接セレクタを繋げることで指定できます。

以下のようなHTMLがあったとすると

```
<p class="current">1番目を基準</p>
<p>2番目</p>
<p>3番目を指定</p>
```

```
.current + p + p {
  font-weight: bold;
}
```

HTMLに改修が入るとCSSセレクタに影響が出るのでなるべくこの実装は避けたいところですね。
