まずは`http://babun.github.io/`からBabunをインストールしてください。
 インストール後、解凍してフォルダ内の`install.bat
`を実行してください。

```
install.bat
[babun] Installing babun
[babun] WARN: Windows HOME environment variable is set to: C:\Users\Uichi
[babun] WARN: ---------------------------------------------------------------
[babun] WARN: FULL COMPATIBILITY CANNOT BE GUARANTEED WHEN 'HOME' IS SET
[babun] WARN: YOU MAY RUN INTO MANY ISSUES THAT CANNOT BE FORESEEN...
[babun] WARN: ---------------------------------------------------------------
[babun] WARN: It's recommended to remove the HOME variable and try again.
[babun] WARN: If you are running the installer from a cmd.exe - restart it.
[babun] WARN: Otherwise the ENV variables will not be propagated to cmd.exe
Do you really wish to proceed (Y / N)?
```

そのままYで進んでださい。

```
[babun] Adding babun to the system PATH variable
[babun] Creating a desktop link
[babun] Babun installed successfully. You can delete the installer now.
[babun] Enjoy @tombujok
[babun] Starting babun
[babun] Starting babun
```

## tmuxのインストール
ターミナルをより使いやすくすることができるtmuxをインストールします。
 tmuxは端末多重化ソフトと呼ばれ、仮想のターミナルをいくつも作り出すことができます。軽くメリットを挙げておくと、一つの画面を分割して複数の仮想端末を作ることや、実行中の処理が途切れないなどがあります。

## 日本語化
Babunはそのままだと日本語が文字化けしてしまうので、日本語化のためのパッケージをインストールします。

```
pact install getent libiconv libiconv-devel inetutils
mkdir ~/tmp
cd ~/tmp
git clone http://github.com/vmi/cocot
cd cocot
./configure &amp;&amp; make &amp;&amp; make install
```

以上でWindowsで使えるイケてるターミナルの構築は完了です。
