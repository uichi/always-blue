ファビコンの設定は、

```
<link rel="icon" type="image/png" href="/images/favicon.ico/">
```

とするのがルールになってます。

ちょっと前までは

```
<link rel="shortcut icon" type="image/png" href="/images/favicon.ico/">
```

このようにrel属性でshortcut iconとするのがルールだったそうです。

ちなみにtype属性ではファイルの形式を選択できます。
