シンタックスハイライトを入れてみました。
定番のPrismを導入しています。
本サイトではMDXというMarkdown変換のプラグインを使用しているのですが、MDXと併用したPrismの利用事例が少なく、やや手こずりました。

導入にはこちらの記事を参考にしました。

[Adding Syntax Highlighting to Gatsby MDX with Prism](https://codetrain.io/adding-prism-syntax-highlighting-to-gatsby-mdx)

## 使ってみた感じ

**JavaScriptコード**

```javascript
const text = 'Hello World!'
console.log(`${text}`)
```

**Pythonのコード**

```python
text = 'Hello World!'
print(text)
```

**Rubyのコード**

```ruby
text = 'Hello World!'
puts text
```

Rubyがハイライトされてなさそう...要調査

サポートされている言語のハイライトはこちらの記事でチェックできます。

[List of supported languages by Prism syntax highlighter](https://lucidar.me/en/web-dev/list-of-supported-languages-by-prism/)
